신분증을 들고 다니지 마세요!

간단한 등록 절차를 거치고, 어플리케이션을 통하여 자신이 성인임을 보이세요.

사진을 찍고, 등록한 뒤, 사용하기까지 1분도 채 걸리지 않습니다.

공인된 본인인증 모듈로 안심하고 사용하세요.

전국 어디서나 빠르고 편리하게 사용할 수 있습니다.

민쯩이는 당신이 지갑에서 신분증을 꺼낼 수고를 덜어줍니다.

기능 소개

가장 중요한 기능만을 담아서 여러분께 선보입니다.

신분증 등록

다양한 유형의 신분증을 등록하고 바로 사용하세요. 등록된 신분증을 메뉴 화면에서 선택하면 진위검사가 진행되고, 검사 결과를 바로 확인할 수 있습니다.

주변 상점 검색

주변에 민쯩이 서비스를 지원하는 상점이 있는지 검색해보세요. 상점에 대한 정보도 확인할 수 있습니다.

개인정보 보호

민쯩이는 사용자의 개인정보를 안전하게 보호합니다. 암호를 설정하여 신분증 내의 식별번호와 같은 민감정보를 타인에게 노출시키지 마세요.

핵심 기술 소개

최선의 기술을 모아서 최고의 가치를 만들고 최상의 경험을 선사합니다.

  • Google Vision OCR

    신분증에 표기된 정보를 인식하고 문자 정보로 변환합니다. 사용자는 개인정보를 직접 입력하지 않아도 됩니다.

  • 경찰청 운전면허증 API

    경찰청에서 제공하는 운전면허증 진위여부조회 API를 사용하여, 경찰청 DB 데이터를 기반으로 운전면허증의 진위를 판별합니다.

  • Google Maps

    현재 사용자의 위치를 표시해주고, 이를 기반으로 사용자는 주변에 본 서비스를 지원하는 상점들을 검색할 수 있습니다.

  • Google Firebase DB

    각 사용자의 개인별 민감 정보를 계정 별로 저장하는 데에 사용합니다. 또한 본 서비스를 지원하는 상점 데이터 저장에도 사용됩니다.
