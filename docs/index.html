이런 경험 있으신가요?

오랜만에 만난 친구와 함께하는 술자리, 대학교 개강 시즌에 맞추어 식당에서는 각 테이블마다 신분증 검사가 한창이다. 그런데 급한 과제를 마치고 나오는 와중에 신분증을 챙기는 것을 깜빡한 당신! 집으로 되돌아가기에는 너무 멀고, 이미 앞에는 주문한 음식들이 나온 상태인데, 친구를 뒤로 하고 이대로 식당을 나서야 할 것인가?

지금 당신에게는 민쯩이가 필요합니다!

신분증을 들고 다니지 마세요!

간단한 등록 절차를 거치고, 어플리케이션을 통하여 자신이 성인임을 보이세요.

사진을 찍고, 등록한 뒤, 사용하기까지 1분도 채 걸리지 않습니다.

공인된 본인인증 모듈로 안심하고 사용하세요.

전국 어디서나 빠르고 편리하게 사용할 수 있습니다.

민쯩이는 당신이 지갑에서 신분증을 꺼낼 수고를 덜어줍니다.

기능 소개

가장 중요한 기능만을 담아서 여러분께 선보입니다.

신분증 등록

다양한 유형의 신분증을 등록하고 바로 사용하세요. 최초 등록시에 신분증을 촬영하게 됩니다. 사용자가 기입한 정보와 사진에서 추출한 데이터가 일치하는지 확인합니다.

유효성 인증

신분증을 사용할 때마다 실시간으로 유효성 검사가 진행됩니다. 신분증이 효력을 상실하면 더 이상 사용할 수 없게 됩니다.

주변 식당 검색

주변에 민쯩이 서비스를 지원하는 식당이 있는지 검색해보세요. 식당에 대한 정보도 확인할 수 있습니다.

개인정보 보호

민쯩이는 사용자의 개인정보를 암호화하여 안전하게 보호합니다. 앱에 비밀 번호를 설정하여 민감정보를 타인에게 노출시키지 마세요.

핵심 기술 소개

최선의 기술을 모아서 최고의 가치를 만들고 최상의 경험을 선사합니다.

  • Google ML Vision OCR

    사진의 내용을 추출하여 문자 정보로 변환하고, 사용자가 입력한 정보와 일치하는지 확인합니다.

  • Alamofire/Kanna

    경찰청 웹 페이지와 HTTP 통신을 진행, 진위여부 결과를 파싱합니다. 경찰청에서 제공하는 운전면허증 진위여부조회 기능을 사용합니다.

  • Google Maps

    사용자 주변에 본 서비스를 지원하는 식당들을 마커로 표시해주고, 식당의 세부 정보를 볼 수 있습니다.

  • Firebase Database/Storage

    민쯩이 서비스와 제휴된 식당들의 데이터를 보관합니다.

  • SQLite3

    (사용자 개인 정보를 암호화하여 로컬에 [미완])
